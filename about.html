CLAX

PORTAL

Es una aplicacion creada, en principio, para empresas de logistica. Sin embargo, el potencial es mayor.

CLAX es una aplicacion de roles y avisos, diseñada para mejorar la productividad y la colaboracion en equipos de trabajo.

ROLES: JEFE, COMANDANTE, SOLDADO

Los roles operan por tropas, y cada rol tiene diferentes permisos

JEFE

El jefe puede cambiarle el nombre a la Tropa, cambiarle el nombre al Comandante y Soldado dentro de la Tropa. Y cuenta con el permiso de disolver/eliminar la Tropa. Puede invitar y asignar roles (JEFE, COMANDANTE, SOLDADO) a los miembros de la Tropa.

COMANDANTE

El comandante puede cambiarle el nombre al Soldado dentro de la Tropa, puede crear tareas y asignarselas al Soldado. No puede eliminar la tropa, No puede cambiarle el nombre a la tropa. Puede Invitar a nuevos miembros a la Tropa y asignarle el rol de COMANDANTE o SOLDADO.

SOLDADO

El soldado puede ver las tareas que le asigno el comandante. Puede invitar a otros miembros como SOLDADO

USUARIO

El usuario no tiene rol por si solo puede crear una tropa y/o unirse a una existente. Puede pertenecer a mas de una Tropa con diferentes roles.

QR y su Potencial

A la hora de invitar a un usuario a la tropa, es sencillisimo, simplemente es un link o QR. y ya esta adentro con Nombre y Apellido, con su rol asignado.

Si necesitas mas, las alertas se pueden programar, se puede ver la cantidad de alertas que tiene cada soldado en cada dia, ya que hay calendario.

Diagrama de soldados

Aunque no sea programable, es facil hacer un diagrama, basicamente, hay un cuadro del mes, y como filas los nombres de los soldados, a cada celda se puede pintar facilmente de un color, y cada vez que se programa un aviso se suma uno +1 a la celda dia-soldado

Pueden haber mas de un diagrama

Los demas diagramas no tienen numero de avisos solo colores.

MAPAS

Se puede integrar con google maps, y que cada soldado asignarle en la alerta una locacion, colocarles alarmas, opcion de compartir su ubicacion en tiempo real.

Una funcion que puede ser util es el del calculo de tiempo en auto de un punto a otro, ya que se le puede poner una alarma para que llegue a tiempo al destino, si, esta pensado para logistica.

MarkDown

Los avisos se pueden hacer en formato MarkDown o normalmente, permitiendo una mayor personalización y organización de la información.

Existe un rol mas: Cliente

Esta opcion esta disponible para Empresas/Tropas Premium, la empresa premium podra compartir su ubicacion en tiempo real con el cliente, a traves de un link. Y el cliente podra ver, solo, las ubicaciones de los soldados solicitados

Es decir, un código de seguimiento

Cliente: Hola, ¿sabe por donde está el "soldado"?

Entiendase por "soldado": transporte, pedido, taxi, etc.

Empresa: Si, mire este link ######

Cliente: Ah, perfecto, muchas gracias!
